#Rajasthan-Turisom-Website
This is a static website designed to promote Rajasthan tourism, built using HTML, CSS, and JavaScript. The website offers a fast-loading and straightforward user experience while showcasing the iconic destinations and cultural richness of Rajasthan through visually appealing designs.
<br><br>

The site provides information about Rajasthan's historical landmarks, traditional festivals, and local cuisines with smooth navigation and an engaging layout. Interactive elements like hover effects and animations, powered by CSS and JavaScript, enhance the overall appeal and make the website more attractive
